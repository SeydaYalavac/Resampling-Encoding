TR:
Veri Yeniden Örnekleme ve Kodlama Teknikleri (Resampling & Encoding)
Bu proje, makine öğrenmesi modellerinde sıkça karşılaşılan "dengesiz veri seti" ve "kategorik değişken" problemlerine odaklanmaktadır. Modelin tarafsız ve yüksek performansla çalışması için gerekli olan teknik yaklaşımları içerir.

Neler Yapıldı?

Dengesiz Veri Yönetimi (Resampling): Azınlık sınıfını artırma (Oversampling) ve çoğunluk sınıfını azaltma (Undersampling) tekniklerinin uygulanması.

Kategorik Veri Dönüştürme (Encoding):

One-Hot Encoding: Nominal veriler için.

Label & Ordinal Encoding: Sıralı veriler için.

Target Encoding: Kategori sayısının çok olduğu durumlar için.

Performans Analizi: Farklı tekniklerin model kararlılığı üzerindeki etkilerinin gözlemlenmesi.

ENG:
Data Resampling & Encoding Techniques
This project focuses on two major challenges in machine learning: "imbalanced datasets" and "categorical variable" management. It provides the necessary technical approaches to ensure models remain unbiased and perform effectively.

Key Features:

Imbalanced Data Management (Resampling): Implementation of Oversampling (increasing the minority class) and Undersampling (decreasing the majority class) techniques.

Categorical Data Transformation (Encoding):

One-Hot Encoding: For nominal data.

Label & Ordinal Encoding: For ordinal data.

Target Encoding: For high-cardinality features.

Performance Impact: Observing how different preprocessing techniques affect model stability and accuracy.
